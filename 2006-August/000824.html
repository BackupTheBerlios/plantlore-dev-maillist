Stavajici zpusob osetrovani vyjimek DBLayerEx a RemoteEx:

try {
     ...
}
catch (RemoteException ex) {
   logger.error("...");
   ex.printStackTrace();
   DefaultReconnectDialog.show(view,ex);
   return;
}
catch (DBLayerException ex) {
   logger.error("...");
   ex.printStackTrace();
   DefaultReconnectDialog.show(view,ex);
   return;
}

Doporuceno nahradit nasledujicim kodem:

try {
     ...
}
catch (Exception ex) {
   logger.error("...");
   // FIXME:
   ex.printStackTrace();
   DefaultExceptionHandler.handle(view, ex);
   // Alternativne DefaultExceptionHandler.handle(view, ex, title);
   return;
}
