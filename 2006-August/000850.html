Jak pouzivat buildscript:

Na prikazove radce sveho shellu se prepnete do korenoveho adresare s
plantlore zdrojaky, tj. u me to napr. bude: 
/home/fraktalek/cvs/plantlore

pak uz muzete spoustet jednotlive ant targety.

Jake targety v build.xml skriptu existuji se dozvite prikazem:

$ ant -projecthelp
Buildfile: build.xml

Main targets:

 build       Creates a new build. Builds both Plantlore client and
server.
 clean       Removes the build and dist directories.
 compile     Compiles the Plantlore client and the server. Including the
necessary RMI compile.
 run-client  Runs the Plantlore client. Compiles first if necessary.
 run-server  Runs the Plantlore server. Compiles first if necessary.
Default target: all


Cili, pokud chcete vytvorit build (spustitelne jary), tak napisete:

$ ant build

Tim se vam vytvori adresar build obsahujici zkompilovane classy
a adresar dist, ktery bude obsahovat vysledne jary a dalsi potrebne
soubory (knihovny, java.policy, atp.).

Jary pak jdou spustit jako normalne:
java -jar plantlore-client.jar (z adresare dist)

Anebo primo pomoci build skriptu:
$ ant run-client

Je potreba mit nainstalovany Apache Ant verze alespon 1.6.2 myslim. Ja
mam 1.6.5 a na nem to funguje urcite.
