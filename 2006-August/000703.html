Ahojte,

na urtaxu v adresari /home/projects/plantlore/databaze/postgreSQL je backup "prazdne" databaze plantlore_empty_2006_08_14.backup. Samozrejme neni uplne prazdna a obsahuje data v tabulce tHistoryColumn a podle prani jsou uvedeni i uzivatele v tabulce tUser a jejich prava v tabulce tRight.

Pro jistotu znovu postup jak si tuto databazi rozchodit u sebe:
1. v pgAdmin3 si vytvorte novou databazi
2. kliknete pravym tlacitkem na tuto databazi a dejte Restore a nactete aktualni backup
3. v zalozce Login Role ( je ve stromu na urovni uzlu Database) zkontrolujte, zda jsou zde uzivatele, ktere mate v tabulce tUser
        - pokud jste uzivatele neobjevili, tak vas ceka jeste par kroku:
          3.1. klikem mysi vyberte prislusnou databazi
          3.2. zvolte ikonu s napisem SQL
          3.3. zadejte nasledujici prikazy:

                        CREATE USER plant PASSWORD 'masterkey' NOCREATEDB CREATEUSER;
                        CREATE USER krater PASSWORD 'masterkey' NOCREATEDB CREATEUSER;
                        CREATE USER kovo PASSWORD 'masterkey' NOCREATEDB CREATEUSER;
                        CREATE USER fraktalek PASSWORD 'masterkey' NOCREATEDB CREATEUSER;
